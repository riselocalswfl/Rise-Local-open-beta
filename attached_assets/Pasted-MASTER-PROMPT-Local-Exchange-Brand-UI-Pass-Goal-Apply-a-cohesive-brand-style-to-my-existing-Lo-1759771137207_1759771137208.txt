MASTER PROMPT — “Local Exchange” Brand/UI Pass

Goal: Apply a cohesive brand style to my existing Local Exchange MVP without breaking functionality. Keep layout and routes the same. Add a light/dark theme and refactor styles to use shared design tokens.

Brand vibe

“Counterculture, yet put together. Aware and artistic.”
Organic, filmic, indie-market feel; clean, modern UI.

Design tokens (use exactly)

Colors

--le-green: #5B8C5A; (primary)

--le-wheat: #EEDC82; (secondary highlight)

--le-clay: #E67A4E; (accent / CTA)

--le-ink: #2D2D2D; (primary text)

--le-paper: #FAF8F5; (page bg)

Dark mode base: --le-night: #181A18;, text on dark: --le-mint: #DDE6D9;

Elevation & radius

Card radius: 14px

Button radius: 9999px (pill)

Shadow sm: 0 2px 8px rgba(0,0,0,.08)

Shadow md: 0 6px 18px rgba(0,0,0,.10)

Typography (Google Fonts)

Headings: Playfair Display (700)

Body/UI: Work Sans (400/600)

Icon set

Lucide (outline) or Feather; stroke 1.75px

Implementation instructions

Create design system

If using Tailwind, extend the theme in tailwind.config.js with these colors, radii, shadows, and font family tokens.

If not using Tailwind, add a styles/design-tokens.css with :root { /* variables above */ } and a [data-theme="dark"] { /* dark overrides */ }. Import globally.

Add utility classes (or Tailwind components) for .le-card, .le-btn, .le-chip, .le-section, and .le-badge.

Global theming

Add a theme toggle in the header (sun/moon). Toggle data-theme="dark" on <html>.

In dark theme: background --le-night, text --le-mint, desaturate images slightly (CSS filter ~0.9) for a night-market vibe.

Typography

Load Playfair Display + Work Sans from Google Fonts.

Apply: h1,h2,h3 { Playfair Display; letter-spacing: .2px }

Body, nav, buttons: Work Sans. Base size 16px; line-height 1.55.

Navigation/header

Header background: white (light), --le-night (dark).

Logo mark: “LE” in a rounded pill with --le-green background and white text.

Search input with subtle shadow and 10px radius.

Active nav link underlined with a 2px --le-clay bar.

Hero (Fort Myers Spotlight banner)

Use a full-bleed image with a filmic overlay gradient (linear-gradient( to bottom, rgba(24,26,24,.15), rgba(24,26,24,.55) ) on dark; lighter on light theme).

Place a small badge top-left: “Fort Myers Spotlight” using .le-badge (pill, bg --le-green, text white).

Title in Playfair, big and confident; CTA button in --le-clay.

Cards

Vendor/Product/Event cards use .le-card: 14px radius, shadow sm, border 1px solid rgba(0,0,0,.06) (light) or 1px solid rgba(255,255,255,.08) (dark).

On hover: transform: translateY(-2px); box-shadow: var(--shadow-md);.

Chips for categories (e.g., Bakery, Organic) with outline style using --le-green text and a faint green bg rgba(91,140,90,.10).

Buttons

Primary: background --le-clay, text white, pill radius, hover darken by ~6%.

Secondary: outline with --le-green, hover fill rgba(91,140,90,.10).

Destructive not needed now.

Section headers use Playfair + a thin underline rule in --le-wheat.

Loading shimmer for cards (subtle).
